# 📊 Telecom Customer Churn Analysis using Python

## 📌 Project Overview

Customer churn is a major challenge for telecom companies, as retaining existing customers is often more cost-effective than acquiring new ones. This project uses Python to perform **Exploratory Data Analysis (EDA)** on a telecom customer dataset to identify the key factors influencing customer churn and generate actionable business insights.

---

## 🎯 Objective

- Analyze customer demographics and service usage.
- Identify factors contributing to customer churn.
- Discover patterns through data visualization.
- Provide business recommendations to improve customer retention.

---

## 📂 Dataset

The project uses the **Telco Customer Churn Dataset**, which includes information such as:

- Customer demographics
- Service subscriptions
- Contract type
- Payment method
- Monthly & total charges
- Customer tenure
- Churn status

---

## 📊 Analysis Performed

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Churn Distribution Analysis
- Contract & Payment Method Analysis
- Monthly Charges & Tenure Analysis
- Correlation Analysis
- Data Visualization

---

## 🔍 Key Insights

- Customers with **Month-to-Month contracts** have the highest churn rate.
- **Fiber Optic** users are more likely to churn.
- Higher **monthly charges** are associated with increased churn.
- Customers with **shorter tenure** are more likely to leave.
- Long-term contracts improve customer retention.

---

## 💡 Business Recommendations

- Encourage long-term contracts through incentives.
- Improve retention strategies for new customers.
- Review pricing for high-paying customers.
- Enhance service quality for Fiber Optic users.
